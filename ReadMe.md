##<a name="1"/>壹、系統敘述

　　對於一間餐廳而言，除了店內的美食、店員的服務品質以及店內舒適度之外，對於餐廳管理者而言最重要的事，就是在於餐點的控管、服務的品質以及餐廳內部的管理。於是我們以餐點系統作為出發點，設計了一套餐廳管理系統，希望可以有助於管理者方便控管以及提升顧客的用餐品質．其中包含員工管理系統、點餐系統、庫存管理系統、營收管理系統、訂位管理系統以及登入系統等六大系統.
