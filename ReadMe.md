##<a name="1"/>壹、系統敘述

　　對於一間餐廳，除了美味的美食，店員的氣氛以及店內舒適之外，對於業者最重要的，就是在於餐點的管控。於是我們著重於餐點系統上，藉由這個系統，希望可以有助於業者在餐點系統，帶來更佳的管控。
